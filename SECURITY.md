# Security Policy

## Supported Versions

We currently only support the latest version of `kpm`.

| Version | Supported          |
| ------- | ------------------ |
| 0.0.1   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within `kpm`, please report it by opening a GitHub Issue or contacting the maintainers directly. We aim to address critical security issues as quickly as possible.
